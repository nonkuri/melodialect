# Melodialect

**作曲家の「音楽的方言」をルールベースで再現する楽曲ジェネレーター**

機械学習を使わず、音楽理論に基づく明示的なルール(モーダル・インターチェンジ、半音階クリシェ、跳躍確率など)をコードとして実装し、コード進行+メロディ+伴奏を生成するブラウザアプリです。生成した曲は MIDI で書き出して DAW に取り込めます。

## 特徴

- **14 のダイアレクト** — 従来の Chromatic / Modal / Pedal / Twilight / Angular / Orchestral / Bossa / Ostinato / Serene に、Flow(6/8)、Blue(12小節ブルース)、Lament(和声的短音階)、Interlock(3-3-2)、Voicing(9th和音と声部連結)を追加
- **8 層の作曲分析** — コード進行、リズム語彙、モチーフ反復、実働する旋律輪郭、非和声音、セクション対比、転調に加え、ダイアレクト固有の音階・グルーヴ・推奨伴奏まで定義
- **合作モード** — 「Verse は Modal、Chorus は Twilight」のようにセクション別へダイアレクトを割り当て可能
- **コード制約生成** — コード原案を固定したメロディ／伴奏生成、空欄補完、リハーモナイズ候補と原案復元、キー相対テンプレートに対応
- **曲全体の作曲設計** — 選択モチーフの固定、Chorus間の変奏量、セクション別の緊張度・密度・明るさ・終止形を設定し、対比を可視化
- **編集可能な仮歌詞** — 日本語／英語／スキャットの再生成、直接編集、音節割り当て、MusicXML／テキスト反映に対応
- **ユーザー定義ダイアレクト** — JSONの項目別検証、安全な値域・サイズ・技法制限、ブラウザ保存と再書き出しに対応
- **拍子対応** — 4/4 / 3/4(ワルツ伴奏)/ 6/8(複合2拍子)
- **カスタム構成** — 標準プリセット7種に加え、`i,v,c,b,c,o` のような構成をユーザーが追加・保存可能(ブラウザに永続化)
- **エンディング/ループモード** — 通常はダイアレクト別の終止形+終止和音を保持するコーダで締める。ループモードでは半終止のまま曲頭へ戻るシームレスな継ぎ目を生成し、再生もリピートする
- **プロジェクト管理** — 曲名・設定・生成結果・手動編集を自動保存し、JSON 入出力と最近使った曲の復元に対応
- **部分再生成とロック** — セクション/パート単位で変奏し、採用したセクション・小節・コード・ノートを保護
- **直接編集** — ピアノロールで複数選択、コピー、移調、クオンタイズ、ベロシティ編集を行い、コードの置換・挿入・削除、Undo/Redo が可能
- **作曲用トランスポート** — シーク、一時停止、区間ループ、メトロノーム、カウントイン、キーボード操作に対応
- **伴奏・ミキサー** — ピアノパターン、ギター、簡易ドラム、5パートのミュート/ソロ/音量/パン/内蔵音色・SoundFontを編集し、Masterを含むミックスをプリセット保存・復元
- **構成・作曲コントロール** — セクションの並べ替え/追加/複製/削除、個別BPM/移調、音域/密度/緊張度/明るさ等を調整
- **譜面表示** — メロディ+コードネームのリードシート(ピアノロールと切替、仮歌詞表示可)
- **WAV / WAVステム / MIDI / MusicXML / テキスト書き出し** — DAW、楽譜ソフト、外部音楽サービスへ用途別に受け渡し
- **ルールベース生成** — 学習データ不要・著作物の複製なし。生成されるのは技法を模した完全オリジナル曲
- **生成根拠の解説** — 「3小節目: ♭VII はモーダル・インターチェンジ」のように、各小節に適用されたルールを表示
- **可変レイアウト** — 編集パネルとピアノロール、ピアノロールと生成根拠の仕切りを上下へドラッグ可能
- **再現性と候補生成** — シードを指定して結果を再現可能。現在の曲と同じシードで全体生成を繰り返すと、新しい候補用のシードへ自動更新
- **MIDI (SMF) 書き出し** — Logic Pro などの DAW にそのまま取り込み可能
- **完全クライアントサイド** — サーバー不要。ブラウザだけで動作

## 動作環境

- Node.js 22 以上(npm 同梱)
- モダンブラウザ(Chrome / Edge / Safari / Firefox 最新版)

## セットアップ

PowerShell にそのままコピペで実行できます(1回だけ必要)。

```powershell
git clone https://github.com/nonkuri/melodialect.git
cd melodialect
npm install
```

## 起動(ブラウザアプリ)

```powershell
npm run dev
```

起動後、ブラウザで **http://localhost:5173** を開くと UI が表示されます。

最初の1曲は次の手順で作れます。

1. 左側でダイアレクトを選ぶ
2. キー、調性、拍子、テンポ、構成を調整する
3. 結果を再現したい場合はシードを指定する
4. コード原案やモチーフを使う場合は **作曲設計 v0.9** で設定する（通常は省略可）
5. 右上の **♪ 全体生成** を押す
6. **▶ 再生** で確認し、伴奏・ミキサー・作曲パラメーターを調整する
7. MIDI、WAV／WAVステム、MusicXML、テキストのいずれかで書き出す

詳しい操作は [ユーザーガイド](docs/USER_GUIDE.md)、設定値の意味は
[パラメーターリファレンス](docs/PARAMETERS.md) を参照してください。

- **♪ 全体生成** — 設定パネル(ダイアレクト・キー・テンポ・構成・シード)の内容で曲を生成。同じシードで再実行した場合は新しいシードへ進み、ボタンに生成完了を表示
- **部分再生成** — タイムラインでセクションを選び、全体/メロディ/コード/伴奏だけを変奏
- **作曲設計 v0.9** — コード制約、モチーフ、Chorus変奏、緊張度カーブ、終止形、仮歌詞、ユーザーダイアレクトを編集
- **構成編集** — 構成バーでセクションを追加・複製・削除。下部タイムラインをドラッグして並べ替え
- **編曲・作曲調整** — 伴奏アレンジと作曲パラメータは編集後に「適用」、または表示中の値で全体生成。ミキサーは即時反映。上下の仕切りで作業領域を調整
- **▶ 再生** — 任意位置から試聴。セクションをダブルクリックすると範囲ループを設定
- **ピアノロール編集** — ノートの追加・移動・伸縮、複数選択、コピー、移調、クオンタイズ、ベロシティ編集、ズーム、再生位置追従に対応
- **MIDI 保存** — 生成した曲を .mid ファイルとしてダウンロード
- **WAV 保存** — ブラウザ内でレンダリングした音声を .wav でダウンロード
- **テキスト出力** — Suno 等に貼り付けるスタイルプロンプト+仮歌詞+コード進行の .txt

終了するにはターミナルで `Ctrl+C` を押します。

## CLI で MIDI を生成する

ブラウザを開かずにコマンドラインだけで曲を生成し、`out\` フォルダに .mid ファイルを書き出します。

```powershell
npm run generate -- --seed 42
```

オプションを指定する例(キー G・テンポ 110・構成 Verse-Chorus-Bridge-Chorus):

```powershell
npm run generate -- --seed 123 --key G --bpm 110 --form v,c,b,c
```

| オプション | 意味 | 既定値 |
|---|---|---|
| `--seed` | 乱数シード(同じ値なら同じ曲) | 42 |
| `--key` | キー(C, Db, D, ... B) | ダイアレクトの推奨キー |
| `--bpm` | テンポ(40〜200) | ダイアレクトの推奨値 |
| `--form` | 構成(v=Verse, c=Chorus, b=Bridge, i=Intro, o=Outro をカンマ区切り)。`v:modal,c:twilight` でセクション別ダイアレクト指定(合作モード) | v,c,v,c |
| `--meter` | 拍子(4/4, 3/4, 6/8) | ダイアレクトの推奨拍子 (通常 4/4、serene は 3/4、flow は 6/8) |
| `--dialect` | ダイアレクト名(chromatic / modal / pedal / twilight / angular / orchestral / bossa / ostinato / serene / flow / blue / lament / interlock / voicing) | chromatic |
| `--loop` | ループモード(半終止で終わり曲頭へ戻る。コーダなし) | なし (終止+コーダ) |
| `--out` | 出力先フォルダ | out |

生成された .mid は Windows Media Player や DAW で再生できます。

## プロジェクト構成

```
src/
  engine/     生成エンジン (UI 非依存の純粋ロジック)
  dialects/   ダイアレクト定義 (作曲スタイルの JSON)
  audio/      Web Audio 再生
  export/     MIDI (SMF) エンコーダ・ダウンロード
  ui/         React コンポーネント
scripts/      検証用 CLI
test/         Vitest テスト
```

## 注意事項

本アプリは実在アーティストの録音・歌詞・メロディを一切収録していません。実装しているのは公知の音楽理論技法と統計的傾向のみであり、生成された楽曲の権利は生成したユーザーに帰属します。ダイアレクト名は技法名に基づく抽象的な名称であり、実在の人物・団体を指すものではありません。
