# Melodialect パラメーターリファレンス

対象バージョン: v1.0.0

[READMEへ戻る](../README.md) | [ユーザーガイド](USER_GUIDE.md)

この文書は、ブラウザ画面にある設定の意味、値を小さく／大きくしたときの変化、反映タイミング、組み合わせ例をまとめたものです。

## 1. 最初に知っておくこと

### 値はダイアレクトに対する相対調整

作曲パラメーターは、すべての曲を同じ形へ変える絶対値ではありません。選択中のダイアレクトが持つコード語彙、リズム、跳躍確率、構成傾向を基準にして強めたり弱めたりします。そのため、同じ `70` でも Chromatic と Twilight では結果が異なります。

### 反映タイミング

| 設定の場所 | 反映タイミング | 生成済みノート |
|---|---|---|
| 左ペインのダイアレクト、キー、調性、拍子、テンポ、構成、シード、ループモード | **♪ 全体生成** を押したとき | 曲全体を作り直す |
| 構成バーの種類、ダイアレクト、小節、移調、BPM、追加／複製／削除／並べ替え | 変更時 | 曲全体を作り直す |
| 伴奏アレンジ | 編集後に **適用** または **♪ 全体生成** | 影響する伴奏ノートを作り直す |
| 作曲パラメーター | 編集後に **適用** または **♪ 全体生成** | 影響するメロディやコードを作り直す |
| ミキサーのM、S、Vol、Pan、音色 | 変更時 | ノートは変えず再生音だけ変える |
| 部分再生成 | ボタンを押したとき | 選択セクションと対象パートだけ変える |

細かなノート編集を始める前に、伴奏アレンジと作曲パラメーターを決めるのがおすすめです。

### 画面の数値表示

- スウィング、ヒューマナイズ、作曲パラメーターは画面上で `0〜100` と表示され、内部では `0.00〜1.00` として扱われます。
- ベロシティは `0.50〜1.50` の倍率です。
- ミキサーのVolは `0〜1.5`、Panは `-1〜+1` です。

## 2. 基本生成設定

| 項目 | 範囲／選択肢 | 意味 |
|---|---|---|
| ダイアレクト | 14種類 | コード語彙、旋律、リズム、構成、推奨伴奏をまとめた作曲スタイル |
| キー | C〜Bの12音 | 曲の主音。メロディ、コード、ベース全体を移調する基準 |
| 調性 | メジャー／マイナー | 使用する音階とローマ数字コードの解釈 |
| 拍子 | 4/4、3/4、6/8 | 1小節の拍構造。伴奏とドラムパターンも変わる |
| テンポ | 40〜200 BPM | 曲全体の基本速度 |
| 構成 | プリセット／カスタム | Intro、Verse、Chorus、Bridge、Outroの並び |
| ループモード | オフ／オン | 曲末を完全終止ではなく半終止にし、曲頭へ戻りやすくする |
| シード | 整数 | 同じ設定と同じシードから同じ曲を再現する |

### ダイアレクトを変更したとき

ダイアレクトを選ぶと、そのダイアレクトの推奨キー、調性、BPM、拍子、伴奏アレンジへ切り替わります。これらは出発点なので、選択後に自由に変更できます。

### キーと調性

キーは音の高さの中心、調性はメジャー／マイナーの音階体系です。歌いやすさを変えたいだけならキーを変更し、曲の明暗を根本から変えたい場合は調性も変更します。

### 拍子

- **4/4** — 一般的なポップ、ロック、ブルース向け
- **3/4** — 1小節を3拍で感じるワルツ。Sereneの推奨拍子
- **6/8** — 8分音符6つを大きく2拍で感じる複合拍子。Flowの推奨拍子

拍子を変更するとメロディのリズムだけでなく、ピアノ、ベース、ギター、ドラムの配置も変わります。

### シード

シードは乱数列の開始値です。同じダイアレクト、設定、構成、シードなら同じ結果を再現できます。曲の性格を保ったまま別案を探すときはシードだけ変えます。🎲は `0〜999999` のランダムなシードを設定します。

ブラウザUIでは、現在の曲と同じシードのまま **♪ 全体生成** を繰り返すと、新しい候補を得られるようシードが自動的に別の値へ進みます。現在と異なるシードを手入力または🎲で選んだ場合は、その指定値を使用します。CLIの `--seed` は常に指定値を使うため、同じ条件なら決定的に再現できます。

## 4. 構成パラメーター

最下段でセクションを選択し、上部の構成バーから設定します。

| 項目 | 範囲 | 効果 |
|---|---|---|
| 種類 | Intro／Verse／Chorus／Bridge／Outro | セクション固有のフレーズ長、密度、カデンツなどの規則を選ぶ |
| ダイアレクト | 14種類 | 選択セクションだけ別の作曲スタイルにする |
| 小節 | 1〜32 | セクションの長さ |
| 移調 | -12〜+12半音 | そのセクションだけ上下へ移調する。+12は1オクターブ上 |
| BPM | 40〜240 | そのセクションだけテンポを変える |

急激な移調やBPM変更は強い対比になります。自然につなげたい場合は、移調を±2〜5半音程度、BPM差を前後10%程度から試してください。

## 5. 伴奏アレンジ

伴奏アレンジの値は編集用として保持され、**適用** を押すと同じシードとコードを基準に影響する伴奏ノートが再構築されます。未適用のまま **♪ 全体生成** を押した場合も、表示中の伴奏値を使って生成し、その値を適用済みにします。**キャンセル** で適用済みの値へ戻し、**推奨値に戻す** で選択ダイアレクトの初期値を編集値へ読み込めます。

### ピアノ

| パターン | 内容 |
|---|---|
| なし | ピアノパートを生成しない |
| ブロック | 拍子に合わせた持続和音。4/4では主に2拍単位 |
| アルペジオ | コード構成音を0.5拍ごとに順番に鳴らす |
| ボサノヴァ | ルートを省いた3rd・7th・9th中心の薄い和音を、ギターと異なる2小節型で鳴らす |
| 8分刻み | コード全音を0.5拍ごとに刻む。密度と推進力が高い |
| バラード | 2拍単位の和音を少しずつずらし、上下の並びを交互にする |
| シンコペーション | 裏拍を中心に短い和音を置く |
| 声部連結 | 前のコードに近い転回形を選び、声部移動を小さくする |

### ギター

| パターン | 内容 |
|---|---|
| なし | ギターパートを生成しない |
| ストラム | コード構成音をわずかにずらして和音として鳴らす |
| アルペジオ | コード構成音を0.5拍ごとに鳴らす |
| ボサノヴァ | 低音を省いた拡張和音を、2小節で変化する裏拍中心の型で鳴らす |
| シンコペーション | 裏拍中心の短いストラム |
| インターロック | 3-3-2系の位置へ単音を交差配置する |

### ドラム

| パターン | 内容 |
|---|---|
| なし | ドラムを生成しない |
| ベーシック | キック、スネア、ハイハットの基本パターン |
| ロック | 2拍目・4拍目のスネアと追加キックを強調 |
| ボサノヴァ | ストレート8分、軽いキック、2小節で変化するクロススティック |
| シャッフル | 3連系の後ろ側へハイハットを置く |
| インターロック | 3-3-2系のキックと交差する打音 |

6/8では「なし」を除く各ドラムパターンで、複合2拍子に合う基本配置が優先されます。ベースは独立したパターン選択を持たず、コード進行、拍子、ダイアレクト固有グルーヴに従います。

### スウィング

| 画面値 | 効果 |
|---|---|
| 0 | 8分音符を均等に配置 |
| 20〜40 | 軽い後ノリ |
| 50〜75 | はっきりしたシャッフル感 |
| 100 | 裏の8分音符を最大0.16拍遅らせる |

スウィングはピアノ、ベース、ギター、ドラムへ適用されます。Blueのように元からシャッフルを持つダイアレクトでは、強くしすぎるとリズムが重くなることがあります。

### ヒューマナイズ

| 画面値 | 効果 |
|---|---|
| 0 | 完全にグリッド通りで均一 |
| 5〜15 | ごく自然なタイミングと強弱の揺れ |
| 20〜40 | 手弾き感が明確 |
| 50以上 | 意図的にラフ。速い曲では乱れやすい |

最大値100では、伴奏ノートの開始位置へ最大約±0.035拍、ベロシティへ最大約±10の揺れを加えます。メロディノートにはこの設定を直接適用しません。

### ベロシティ

`0.50〜1.50` の倍率で、ピアノ、ベース、ギター、ドラムのノート強度をまとめて調整します。

- `0.50` — 元の半分。かなり弱い
- `1.00` — 基準
- `1.50` — 元の1.5倍。上限127でクリップ

これは伴奏ノートの強弱を作り直す値です。最終的な聞こえる音量はミキサーのVolで調整してください。

## 6. 作曲パラメーター

作曲パラメーターは画面上で `0〜100` です。中央の `50` が基本状態です。変更値は **適用** または **♪ 全体生成** を押したときに曲へ反映され、適用前なら **キャンセル** で戻せます。A/Bボタンは変更前後を同じ再生位置で比較します。

### 音域

| 項目 | 範囲 | 意味 |
|---|---|---|
| 最低音 | MIDI 36〜最高音未満 | 生成メロディの下限 |
| 最高音 | 最低音超〜MIDI 96 | 生成メロディの上限 |

目安となるMIDI音番号は、C2=`36`、C3=`48`、A3=`57`、C4=`60`、A4=`69`、C5=`72`、C6=`84`、C7=`96` です。初期値は最低音57、最高音84です。

音域を狭くすると歌いやすくまとまりやすくなり、広くするとドラマチックになります。最低音と最高音を狭くしすぎると、異なる生成音が同じ音域へ折り畳まれやすくなります。

### 主要9パラメーター

| パラメーター | 小さい値 | 大きい値 | 実装上の主な効果 |
|---|---|---|---|
| 音数密度 | 音を間引き、余白を増やす | 長い音を分割し、音数を増やす | 48未満で間引き、68超で一部の1拍以上の音を分割 |
| 和声複雑度 | ゆっくりしたコード変化、基本進行寄り | 1小節2コードや固有イディオムが増える | ハーモニックリズムと固有コード技法の確率を調整 |
| 緊張度 | 安定した構成と進行 | 固有イディオム、変則フレーズ、転調が増えやすい | 意外性と組み合わせて構造的な変化量を決める |
| 跳躍 | 順次進行中心 | 5度前後以上の音程移動が増える | ダイアレクト固有の跳躍確率を0.5〜1.5倍程度で調整 |
| 反復 | 同音・モチーフの繰り返しが減る | フックや反復モチーフが増える | 同音反復とモチーフ反復確率を調整 |
| シンコペーション | 追加の後ろずらしなし | 一部のメロディ音を裏側へずらす | 50超で効果が始まり、最大約0.225拍ずらす |
| 明るさ | メロディを控えめな強さにする | メロディのベロシティを上げる | 音階やコードの長短ではなく、主にメロディの発音強度 |
| 穏やかさ | 跳躍とメロディの強さが増える | 跳躍とメロディの強さを抑える | 跳躍確率とメロディベロシティを同時調整 |
| 意外性 | 規則的で予測しやすい | 変則フレーズや転調が増えやすい | 緊張度との平均で構造変化の確率を調整 |

### 音数密度の段階

- `0〜19` — おおむね3音に1音を残す強い間引き
- `20〜47` — おおむね2音に1音を残す
- `48〜68` — 元のリズムをほぼ維持
- `69〜100` — 一部の長い音を2つへ分割

音数密度は連続的というより、上記のしきい値で変化が大きくなります。

### 緊張度と意外性

この2つは一部の処理で組み合わされます。

- **緊張度** はコード固有技法の出現確率にも直接影響します。
- **意外性** と **緊張度** の平均が、変則フレーズと転調の増減に使われます。
- 和声だけ複雑にしたい場合は和声複雑度を上げ、構成上の予想外も欲しい場合は意外性を上げます。

### 跳躍と穏やかさ

跳躍を上げても、穏やかさが高いと跳躍は抑えられます。大きく印象的なサビを作りたい場合は跳躍を上げ、穏やかさを中〜低にします。滑らかなメロディにしたい場合は跳躍を下げ、穏やかさを上げます。

### 明るさについて

現在の明るさはメジャー／マイナーやコードの明暗を切り替えるものではなく、主にメロディのベロシティを変えるマクロです。和声的な明暗は **調性** と **ダイアレクト** で調整してください。

## 7. ミキサーと音色

ミキサー変更は生成ノートを作り直さず、再生とWAV書き出しへ反映されます。

| 項目 | 範囲 | 意味 |
|---|---|---|
| Master音量 | 0〜1.5 | 全パートをまとめた最終音量 |
| ピーク保護 | オフ／オン | 出力段の過大ピークを抑える |
| M | オフ／オン | ミュート。オンのパートを消音 |
| S | オフ／オン | ソロ。1つでもオンなら、ソロでないパートを消音 |
| Vol | 0〜1.5 | パートの線形音量倍率。1が基準 |
| Pan | -1〜+1 | -1が左、0が中央、+1が右 |
| 内蔵音色 | パート別候補 | オシレーター波形、立ち上がり、減衰、フィルターを変更 |
| SoundFont | SF2のバンク／プログラム | 取り込んだ音源または標準音源のプリセットをパートへ割り当てる |

Master音量は `1.0`（100%）が基準です。内蔵オシレーターとSoundFontの出力差は内部で補正され、初期値の80%ではピーク余裕を確保します。100%を超えて増幅する場合は、ピーク保護をオンにしてください。

### ミキサープリセット

**現在のミックスを保存** は、Master音量とピーク保護、および全パートのM、S、Vol、Pan、内蔵音色、SoundFont割り当てを最大20件までブラウザに保存します。**プリセットを読込…** では、保存した全項目を一度に復元します。プリセットは曲のノートや作曲パラメーターを変更しません。

### 音色の特徴

| パート | 音色 | 傾向 |
|---|---|---|
| メロディ | フルート | 丸い三角波、少し柔らかな立ち上がり、持続感あり |
| メロディ | ソフト | 倍音の少ないサイン波。最も滑らか |
| メロディ | リード | ノコギリ波。輪郭が強く前に出やすい |
| ピアノ | グランド | 短い立ち上がりと自然な減衰 |
| ピアノ | エレピ | サイン波系で柔らかく、グランドより余韻が長い |
| ピアノ | オルガン | 矩形波系で持続音が長い |
| ギター | ナイロン | 三角波系で丸く控えめ |
| ギター | ブライト | ノコギリ波系で倍音が多い |
| ベース | フィンガー | 三角波と1オクターブ下の副音。丸い低音 |
| ベース | シンセ | 矩形波と副音。硬く集中した低音 |
| ドラム | 電子 | 高めのフィルターを通した短い電子打音 |
| ドラム | ブライト | やや持続が長く、倍音感の異なる打音 |

音色ごとの知覚音量差は内部で補正していますが、音域、同時発音数、スピーカーによって聞こえ方は変わります。音色を決めた後にVolで最終調整してください。

### パンの出発点

初期値はピアノが少し左、ギターが少し右、メロディ、ベース、ドラムは中央です。ベースは中央付近を保ち、ピアノとギターを左右へ少し分けると混雑を避けやすくなります。

## 8. 組み合わせ例

数値は出発点です。ダイアレクトの個性に合わせて5〜10ずつ調整してください。

### 歌いやすい王道ポップ

- ダイアレクト: Chromatic
- 音数密度: 50〜60
- 和声複雑度: 40〜55
- 跳躍: 40〜55
- 反復: 65〜80
- 意外性: 20〜40
- 穏やかさ: 45〜60

### 切ないバラード

- ダイアレクト: Twilight または Lament
- ピアノ: バラード
- ギター／ドラム: なし、または控えめ
- 音数密度: 30〜45
- 和声複雑度: 55〜70
- 緊張度: 55〜70
- 明るさ: 30〜45
- 穏やかさ: 65〜80
- ヒューマナイズ: 10〜18

### 印象的で大きなサビ

- セクション: Chorusを選択して調整
- 音数密度: 60〜75
- 跳躍: 70〜85
- 反復: 60〜75
- 明るさ: 65〜80
- 穏やかさ: 25〜45
- 緊張度: 55〜70

### ミニマルで反復的

- ダイアレクト: Ostinato
- ピアノ: 8分刻み
- 音数密度: 15〜35
- 反復: 80〜100
- 跳躍: 10〜30
- 意外性: 10〜30
- ヒューマナイズ: 0〜5

### リズムを前面に出す

- ダイアレクト: Interlock、Bossa、Blue
- ピアノ: Bossaではボサノヴァ、Interlock/Blueではシンコペーション
- ギター: Bossaではボサノヴァ、ほかはシンコペーションまたはインターロック
- ドラム: 対応するボサ、シャッフル、インターロック
- シンコペーション: 65〜85
- スウィング: ダイアレクト初期値から5〜10ずつ調整
- ヒューマナイズ: 5〜15

### 実験的で予想外

- ダイアレクト: Angular または Voicing
- 和声複雑度: 75〜95
- 緊張度: 70〜90
- 跳躍: 65〜85
- 反復: 20〜45
- 意外性: 80〜100

## 9. 調整のコツ

1. 一度に1〜2項目だけ変更し、何が変わったかを確認します。
2. シードを固定すると、パラメーター差を比較しやすくなります。
3. 大きな方向性はダイアレクト、キー、調性、拍子で決めます。
4. 伴奏パターンと作曲パラメーターを先に決め、ノート編集は後にします。
5. 曲の一部だけ変えたい場合は、全体生成ではなく部分再生成を使います。
6. 気に入った候補はバリエーションとJSONの両方へ残します。
7. 音色変更後はVolを再確認します。倍音の多い音色は同じ測定音量でも前に聞こえる場合があります。

## 10. 用語

| 用語 | 意味 |
|---|---|
| ダイアレクト | 作曲上の語彙と傾向をまとめたスタイルプリセット |
| シード | 生成結果を再現するための乱数開始値 |
| ベロシティ | MIDIノートの発音強度。通常1〜127 |
| ハーモニックリズム | コードが変わる頻度 |
| シンコペーション | 強拍を避け、裏拍や拍の間を強調するリズム |
| モチーフ | 繰り返しや変形に使う短い旋律単位 |
| パン | ステレオの左右位置 |
| 半終止 | 曲を完全に閉じず、続きや曲頭への復帰を感じさせる終わり方 |

CLIオプションの指定方法は、[READMEのCLI節](../README.md#cli-で-midi-を生成する)を参照してください。
